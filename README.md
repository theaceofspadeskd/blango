# Blango

Installation:

1. Download this project
    ```
    git clone https://github.com/theaceofspadeskd/blango.git
    ```
2. Install all necessary dependencies
    ```
    pip install -r requirements.txt
    ```
3. Make migrations
    ```
    python manage.py makemigrations
    ```
4. Migrate
    ```
    python manage.py migrate
    ```
